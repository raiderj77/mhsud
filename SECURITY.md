# Security and privacy reports

Please report a suspected security or privacy issue privately through GitHub's
private vulnerability reporting feature when it is available for this
repository. Do not open a public issue containing personal, health, credential,
or exploit details.

If private vulnerability reporting is unavailable, use the contact method on
https://mindchecktools.com/contact and include only the minimum information
needed to arrange a secure follow-up.

## Scope priorities

- exposure or transmission of screening answers, scores, or local records
- caching or restoration of sensitive assessment state
- analytics, advertising, affiliate, or session-replay activity on sensitive routes
- authentication, injection, dependency, or deployment-secret vulnerabilities

Please do not submit real health information, credentials, or destructive proof
of concept data when reporting a problem.
